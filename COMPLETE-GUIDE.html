⚡ Quick Start - Get Running in 5 Minutes

1️⃣ Install Node.js

Go to https://nodejs.org/

Download the "LTS" version and install

⚠️ Important: Download "LTS", not "Current"

2️⃣ Setup Project

Open Command Prompt/Terminal and run:

cd path/to/your/wedding-website-folder npm install

3️⃣ Run Website

In the same terminal, run:

npm run dev

Open browser to: http://localhost:5173

4️⃣ Basic Changes

  • Change "Sarah & Michael" to your names in index.html
  • Update wedding date in main.js
  • Replace venue info in Details section
  • Add your photos to public/ folder

5️⃣ Deploy

npm run build

Upload the dist folder to Netlify, GitHub Pages, or your hosting

📋 Complete Setup Guide
📋 Prerequisites - What You Need

Before You Start

Make sure you have these items ready:

  • ✅ A computer with internet connection
  • ✅ A text editor (download Visual Studio Code for free)
  • ✅ A web browser (Chrome recommended)
  • ✅ Your wedding photos and information
💡 Tip: Visual Studio Code is the best free text editor. Download it from https://code.visualstudio.com/
📥 Step 1: Download and Extract Files

Getting Your Template Files

1. Download the template ZIP file

2. Right-click the ZIP file and select "Extract All"

3. Choose a folder location (like Desktop)

4. Open the extracted folder to see all files

wedding-landing/
├── index.html          # Main website file
├── style.css           # Custom styles
├── main.js             # JavaScript functionality
├── package.json        # Project configuration
├── vite.config.js      # Build configuration
├── public/             # Image folder
│   ├── all.png
│   ├── c-1.png
│   └── ...
└── README.md          # Documentation
✅ Success: You should see all these files in your folder
⚙️ Step 2: Install Node.js (Required)

Installing the Required Software

Node.js is required to run the website. Here's how to install it:

  1. Go to https://nodejs.org/
  2. Click the big green "LTS" button to download
  3. Run the downloaded installer
  4. Follow the installation wizard (click "Next" for all options)
  5. Restart your computer
⚠️ Important: Make sure to download the "LTS" version, not the "Current" version
💡 Tip: After installation, you can verify it worked by opening Command Prompt and typing: node --version
🔧 Step 3: Setup Your Project

Installing Dependencies

Now you need to install the required packages:

  1. Open Command Prompt (Windows) or Terminal (Mac/Linux)
  2. Navigate to your project folder using the cd command
  3. Type this command and press Enter:
npm install

Wait for the installation to complete (this may take a few minutes)

✅ Success: You should see a "node_modules" folder created in your project
🚀 Step 4: Run Your Website

Starting the Development Server

Now you can view your website:

  1. In the same terminal/command prompt, type:
npm run dev

2. Wait for the message "Local: http://localhost:5173"

3. Open your web browser

4. Go to: http://localhost:5173

🎉 Congratulations! You should now see your wedding website running!
💡 Tip: Keep the terminal window open while working on your website. To stop the server, press Ctrl+C
🎨 Customization Guide

1. Change Couple Names

Find and replace all instances of "Sarah & Michael" with your names:

File: index.html

Before

<title>Sarah & Michael - Wedding Website</title> <h1>Sarah & Michael</h1>

After

<title>John & Jane - Wedding Website</title> <h1>John & Jane</h1>
💡 Tip: Use Ctrl+F (or Cmd+F on Mac) to find all instances quickly

2. Update Wedding Date

Change the countdown timer date:

File: main.js (around line 50)
// Find this line and change the date const weddingDate = new Date('2024-06-15T16:00:00'); // Change to your wedding date, for example: const weddingDate = new Date('2024-12-25T14:00:00');
⚠️ Important: Use the format: YYYY-MM-DDTHH:MM:SS (24-hour format)

3. Photo Customization

Recommended Image Sizes:

Hero Background: 1920x1080px (landscape)
Gallery Images: 800x600px (various orientations)
Profile Photos: 400x400px (square)
Story Images: 600x400px (landscape)

4. Color Customization

5. Contact Form Setup

To make the contact form functional:

  1. Go to https://www.emailjs.com/
  2. Create a free account
  3. Set up an email service (Gmail, Outlook, etc.)
  4. Create an email template
  5. Update the configuration in main.js
💡 Alternative: If you don't want to set up EmailJS, replace the contact form with a simple email link:
<a href="mailto:your-email@example.com">Contact Us</a>
🛠️ Troubleshooting Common Issues
❌ Problem: Website not loading
✅ Solution: Check if Node.js is installed by typing node --version in terminal
❌ Problem: Images not showing
✅ Solution: Check if image files exist in the correct folder and file names match exactly
❌ Problem: Styling looks wrong
✅ Solution: Clear browser cache (Ctrl+F5) and check if all CSS files are loaded
❌ Problem: Contact form not working
✅ Solution: Check EmailJS configuration or replace with a simple email link

🚀 Deployment Guide

Building for Production

When ready to deploy:

npm run build

This creates a dist folder with optimized files for hosting.

Deployment Options

🌐 Netlify (Recommended)

1. Go to https://netlify.com/

2. Drag and drop the dist folder

3. Your website will be live instantly!

📚 GitHub Pages

1. Create a GitHub account

2. Create a new repository

3. Upload your files

4. Enable GitHub Pages in settings

🏢 Traditional Hosting

1. Purchase web hosting (GoDaddy, HostGator, etc.)

2. Upload files via FTP

3. Point your domain to the hosting

✅ Success: Your wedding website is now live and ready to share with guests!
💡 Tip: Use Netlify for the easiest deployment - just drag and drop!
